Remote / Mobile Access

Tailscale / VPN mode
Phone connects to HDHR directly over Tailscale subnet routing. Best option — no proxy needed. Mutually exclusive with Remote Relay.
Remote relay (no VPN)
For cellular or networks without Tailscale. Routes streams through your hdhr-xmltv server's ffmpeg proxy. Set your relay URL below.
Your hdhr-xmltv subdomain. Streams are proxied to /proxy/v4.1 etc. for ffmpeg H.264 transcoding.

Mobile Software Decoder Proxy

Android phones can't decode MPEG-2 video natively. Enable this to route HDHR streams through an ffmpeg proxy on your server for H.264 transcoding. Leave off if using Tailscale or a relay URL (both handle transcoding already).

Enable mobile proxy

ℹ How VOD Pre-config Works

These settings are saved in the backup JSON under settings.vodConfig.
When StreamVault imports this backup (via Inject or Network Inject), it reads vodConfig and applies it to the app’s VOD settings — only if the device has no existing VOD config (first-time setup or fresh install).
To force-apply on an existing install, open the VOD Settings page on the device and use Scan Repos for Groups after injecting.
